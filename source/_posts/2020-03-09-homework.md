## 公约数和公倍数

**题目内容：**输入两个正整数m和n，求其最大公约数和最小公倍数

**输入格式:**m n

**输出格式：**最大公约数 最小公倍数

**输入样例：**4 14

**输出样例：**2 28

<!--more-->

使用while循环来实现辗转相除法

```c
#include <stdio.h>

int main() 
{
	int m, n,a,b,c,i;
	scanf("%d %d", &m, &n);
	if (m > n)
	{
		a = m;
		b = n;
	}
	else
	{
		a = n;
		b = m;
	}
	i = a % b;
	while (i!=0)
	{
		a = b;
		b = i;
		i = a % b;
	}
	c = (m * n) / b;
	printf("%d %d", b, c);

	return(0);

}
```

如果用for循环呢？只需将

```c
i = a % b;
	while (i!=0)
	{
		a = b;
		b = i;
		i = a % b;
	}
```

换成

```c
for (i= a%b; i != 0; )
	{
		a = b;
		b = i;
		i = a % b;
	}
```

由于我只用了辗转相除法，所以换成for之后，有种换汤不换药的感觉。其实最大公约数有很多求法，最经典的有：

（1）辗转相除法

（2）相减法

（3）穷举法

**输出格式：**水仙数1 水仙数2 水仙数3 ...

正确答案：

```c
#include <stdio.h>

{
```

## 自由落体

**题目内容：**一个球从100m高度自由落下，每次落地后反跳回原高度的一半，再落下，再反弹。求它在第10次落地时，共经过多少米，第10次反弹多高。(均保留2位小数)

**输出格式：**

第10次落地时共经过xx米

第10次反弹xx米

```c
#include <stdio.h>

int main()
{
	double s = 100, h = 100;
	for (int i = 1; i != 10; i++)
	{
		h = h / 2;
		s = s + h * 2;
	}
	printf("第10次落地时共经过%.2lf米\n", s);
	printf("第10次反弹%.2lf米", h / 2);
	return 0;
}
```

## 反序数

**题目内容：** 编写程序，从键盘输入一个4位正整数，输出该数的反序数。反序数即原数各位上的数字颠倒次序形成的另一个整数，例如1234的反序数是4321，2468的反序数是8642。

**输入格式:**  4位正整数

**输出格式：**反序数

**输入样例：**1234

**输出样例：**4321

```c
#include <stdio.h>
int main()
{
	int x,a,b,c,d;
	scanf("%d", &x);
	a = x / 1000;
	b = (x - a * 1000) / 100;
	c = (x - a * 1000 - b * 100) / 10;
	d = x % 10;
	printf("%d%d%d%d",d,c,b,a);

	return 0;

}
```

## 整数解

**题目内容：**方程`x1+x2+x3+x4=30`有多少满足`x1>=2,x2>=0,x3>=-5,x4>=8`的整数解？（用for循环或while循环实现）

**输出格式：**整数解个数

```c
#include <stdio.h>
int main()
{
	int x1 = 2, x2 = 0, x3 = -5, x4 = 8, i = 0;
	for (x1 = 2; x1 <= 35; x1++)
	{
		if (x1 + x2 + x3 + x4 == 30)i++;
		for (x2 = 0; x2 <= 35; x2++)
		{
			if (x1 + x2 + x3 + x4 == 30)i++;
			for (x3 = -5; x3 <= 35; x3++)
			{
				if (x1 + x2 + x3 + x4 == 30)i++;
				for (x4 = 8; x4 <= 35; x4++)
				{
					if (x1 + x2 + x3 + x4 == 30)i++;					
				}
			}
		}
	}
	printf("%d", i);

	return 0;
}
```

### 注意

1.在for( )语句里边的`表达式1`一定要再次赋值，因为在前面的for语句里边你不知道他变成了什么数。

2.为了减少运行时间，在for()语句里边的`表达式2`中，x小于等于的值不低于35，最好35，因为当`x3=-5`最小值时，这时另外两个数为`0`，要使结果等于30，那么剩下的一个数必须为35。**当然**，按照这个说法，其实只需将一个表达式2的数改为35就可以了，其他for语句里的表达式2都可以改为更小的值，由于时间关系，这里不做分析。

## 选秀评分

**题目内容：**在选秀节目中有10个评委为参赛选手打分，分数为1到100分，选手最后的得分为去掉一个最高分和去掉一个最低分后其余8个分数的平均分，编写程序，输入10个分数，输出选手的最后得分(保留2位小数)。

**输入格式:**十个评委的打分

**输出格式：**最后得分

**输入样例：**9 8 7 7 8 9 4 8 8 7

**输出样例：**7.75

```C

#include <stdio.h>
int main()
{
	double i = 0, max = 100, min = 1, m = 0, a = 0;
	while (i < 10)
	{
		scanf("%lf", &a);
		if (a > min) min = a;
		if (a < max) max = a;
		i++;
		m += a;
	}
	m = (m - max - min) / 8.0;
	printf("%.2lf\n", m);

	return 0;
}

```

### 补充

之前在做这道题的时候看到一段代码，颇为心动，它提供了另外一种思路——但是怎么都过不了，这段代码从哪里来的我已经忘了（如果有人知道请告诉我），原来是什么样子我也忘了，后来迫于时间关系改用了其他方法。下面来欣赏一下这段代码：

```c
#include <stdio.h>
int main()
{
	double score[10];
	int i;
	double min = 1.0, max = 100.0, sum = 0.0;
	for (i = 1; i != 10; i++)
	{
		scanf("%lf", &score[i]);
		if (max < score[i])
		{
			max = score[i];
		}
		if (min > score[i])
		{
			min = score[i];
		}
	}
	for (i = 1; i != 10; i++)
	{
		if (score[i] == max || score[i] == min)
			continue;
		sum = sum + score[i];
	}

	printf("%.2lf\n", sum / 8.0);
	
	return 0;

}
```

这段代码的关键在于`double score[10];`他为重复性极强的需要输入的循环提供了便利，但是为什么过不了呢？

我发现其中一个原因，在第二段`for循环`有以下代码：

```c
if (score[i] == max || score[i] == min)
			continue;
```

这段代码的本意时判断`score集合`的某个值是否是最大最小值，如果是，跳过。看到这里的时候我高兴坏了，世间还有这种实现方法。**但是**，当集合有两个同样的数字，并且这个数字都为最大（最小）值时，程序会怎样？都跳过。

于是在`printf`时， `sum / 8.0`，因为我们只去掉一个最大值和一个最小值，这时少了一个数却同样除以8.0，结果就出现错误。当然，我敢保证，这里边还有其他错误——但我真的找不出来 ：）

**错误更新**（3.16）： 今天学了数组，才发现`double score[10];`是代表0-9.

`int a [10]={1,2,3}`为赋值语法。

### 另一种解法

感谢[张元瑞同学](https://user.qzone.qq.com/1186327108?ADUIN=1228014966&ADSESSION=1584000509&ADTAG=CLIENT.QQ.5689_FriendInfo_PersonalInfo.0&ADPUBNO=26982&source=namecardstar)提供了另外一种解法的代码，这下我终于安心了。(￣∇￣)

```C
#include<stdio.h>
int main()
{
	int i, j, a[10], b;
	double m = 0;
	for (i = 0; i < 10; i++)
	{
		scanf("%d", &a[i]);
	}
	for (j = 0; j < 9; j++)
	{
		for (i = 0; i < 9 - j; i++)
			if (a[i] < a[i + 1])
			{
				b = a[i];
				a[i] = a[i + 1];
				a[i + 1] = b;
			}
	}
	for (i = 1; i < 9; i++)
	{
		m = m + a[i];
	}
	m = m / 8;
	printf("%.2lf", m);
	return 0;
}
```
