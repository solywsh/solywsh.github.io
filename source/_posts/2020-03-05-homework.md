## 求最大值

**输出格式：**max

**输出样例：**3

#include<stdio.h>

## 二元一次方程

**题目内容：**

编写程序，从键盘输入一个二元一次方程ax^2+bx+c=0的三个参数a、b、c（均为整数），求此方程的实根。如果方程有实根，则输出实根（精确到小数点后2位，较大实根在前），如果没有实根，则输出"方程没有实根"。

**输入格式:**a b c

**输出格式：**x1 x2

**输入样例：**1 4 3

**输出样例：**-1.00 -3.00

```c
#include<stdio.h>
#include<math.h>
int main()
{
	double a, b, c, disc, x1, x2, p, q;
	scanf("%lf %lf %lf", &a, &b, &c);
	disc = b * b - 4 * a * c;
	if (disc < 0)printf("方程没有实根");
	else
	{
		p = -b / (2.0 * a);
		q = sqrt(disc) / (2.0 * a);
		x1 = p + q;
		x2 = p - q;
		if (x1 > x2)printf("%.2lf %.2lf",x1,x2);
		else
		{
			printf("%.2lf %.2lf", x2, x1);
		}		
	}

	return 0;
}
```

## 判断闰年

**题目内容：**

写一个程序判断某一年是否为闰年

(①普通年能整除4且不能整除100的为闰年.(如2004年就是闰年,1901年不是闰年)②世纪年能整除400的是闰年.(如2000年是闰年,1900年不是闰年))

**输入格式:**年份

**输出格式：**xx is a leap year    或     xx is not a leap year

**输入样例：**2001

**输出样例：**2001 is not a leap year

```c
#include<stdio.h>
int main()
{
	int Y;
	scanf("%d", &Y);
	if (Y % 400 == 0)
		printf("%d is a leap year", Y);
	else
	{
		if (Y % 4 == 0 && Y % 100 != 0)
			printf("%d is a leap year", Y);
		    else
		    {
			printf("%d is not a leap year", Y);
		    }
	}
		
	return 0;
}
```

### 注意

1.在写 if (Y % 400 == 0) 时，最开始我写的是 if (Y % 400 = 0)，编译器这里再次提醒我表达式必须是可修改左值，当我将‘=’改为'=='就好了，这是因为，在c语言里，'='表示赋值，而'=='才表示赋值。

2.最开始我的代码是这样的：

```c
if (Y % 400 == 0)
		printf("%d is a leap year", Y);
if (Y % 4 == 0 && Y % 100 != 0)
		printf("%d is a leap year", Y);
printf("%d is not a leap year", Y);
```

这里注意，如果没有else，在执行一个if后会继续执行下一个if，在这里，每次三个都会执行，不仅不正确，还降低了效率。当输入400时，结果为：

```
400 is a leap year400 is not a leap year
```
