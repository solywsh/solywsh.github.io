## 注意

注意，在`void replace(int* x)`函数下的交换功能并不完善。

举例：`10 9 1 8 7 6 5 4 3 2`这种就会出现错误结果，输出结果为`2 9 10 8 7 6 5 4 3 1`

原因：这个数组的最大值为10，但是位置是在第一个——也就是最小值需要交换的地方，最大值和最小值的交换不可能同时进行，我的代码时先交换最小值，这就会出现一个情况：最小值与第一位互换位置后，最大值的位置被替换了。

所以我们需要再添加一个`else if`来判断这个条件，但是，作业系统并没有测试这个情况，出于时间考虑，就没继续添加了。毕竟，肝不要钱，但是要命。

**题目内容：** 输入两个整数，然后让用户选择1或2，选1时调用max函数，输出二者中的大数，选2时调用min函数，输出二者中的小数。

**输入格式:**   数一 数二 模式（1或2）

**输出格式：**最大数或最小数

**输入样例：**10 12 2

**输出样例：**10

{% note danger %}
函数的好处就是方便在其他地方使用，在第一题的时候已经写过max，min最大最小的函数了，直接拿过来修改就可以了。
{% endnote %}

```c
#include<stdio.h>
int i, flag;
int main(void)
{
	void input(int* x);
	int formax(int* x);
	int formin(int* x);
	int a[2],choice;
	int* p = a;
	input(p);
	scanf("%d", &choice);
	if (choice==1)
	{
		printf("%d", *(p + formax(p)));
	}
	else if (choice == 2)
	{
		printf("%d", *(p + formin(p)));
	}
	return 0;
}

void input(int* x) //输入
{
	for (i = 0; i < 2; i++)
	{
		scanf("%d", x + i);
	}
}
int formax(int* x)//找最大值
{
	for (i = 1, flag = 0; i < 2; i++)
	{
		if (*(x + flag) < *(x + i))
		{
			flag = i;
		}
	}
	return (flag);
}
int formin(int* x) //找min
{
	for (i = 1, flag = 0; i < 2; i++)
	{
		if (*(x + flag) > * (x + i))
		{
			flag = i;
		}
	}
	return (flag);
}
```
