# 最大公约数和最小公倍数

**题目内容：**

写2个函数，分别求两个整数a、b的最大公约数hcf和最小公倍数lcd。

用主函数调用这两个函数，并输出结果。

注意输入输出均以空格分隔

**输入格式:**   a b

**输出格式：**hcf lcd

**输入样例：**3 4

**输出样例：**1 12

```c
#include <stdio.h>
int main()
{
	int hcf(int x ,int y);
	int lcd(int x, int y);
	int m, n;
	scanf("%d %d", &m, &n);
	printf("%d %d", hcf(m, n), lcd(m, n));

	return 0;

}
int hcf(int x,int y)
{
	int a, b, c, i;
	if (x > y)
	{
		a = x;
		b = y;
	}
	else
	{
		a = y;
		b = x;
	}
	i = a % b;
	while (i != 0)
	{
		a = b;
		b = i;
		i = a % b;
	}
	return (b);
}
int lcd(int x, int y)
{
	int c;
	c = (x * y) / hcf(x, y);
	return (c);

}
```

``` c
#include<stdio.h>
int main() {
    int n, m, s, i, sum = 0;
    int a[20];
    scanf("%d %d", &n, &m);
    s = n;
    for (i = 1; s != 0; i++) {
        a[i] = s % m;
        s = s / m;
        sum++;
    }
    for (i = sum; i > 0; i--) {
```

## 原理

我最初的想法是利用 `%o`，`%x`  输出时直接转换的，但是，这样只能转换8进制`%o`，16进制 `%x` ,题目要求的是2-16进制，其他的进制怎么操作呢？详细的方法可以查看《计算机科学导论》数字系统部分。各种进制的转换都有详细的说明。并且之前也是学过。

我只讲大概的原理，十进制转换为其他进制时，我们拿十进制35转换2进制举例：

开始

35除以2，17余1，2进制倒数第一位是1；

17除以2，8余1，2进制倒数第二位为1；

8除以2，4余0，二进制倒数第三位为0；

4除以2，2余0，二进制倒数第四位为0；

2除以2，1余0，二进制倒数第五位为0；

1除以2，0余1，二进制倒数第六位为1；

结束

结果为：100011

10进制转换为其他进制很简单，原理都是一样的，不断的相除取余直到为0。
